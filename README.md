# Save Pass

Gestor de contrasenas simple hecho en Python con interfaz grafica Tkinter.

## Caracteristicas
- Generacion de contrasenas seguras
- Guardado local de contrasenas en JSON
- Interfaz grafica intuitiva (Tkinter)
- Cifrado basico de datos

## Uso
1. Descarga el ejecutable desde [Releases](https://github.com/Lesternx/save-pass/releases)
2. Ejecuta `Save-Pass.exe`
3. Genera y guarda tus contrasenas

## Tecnologias
- Python 3.14
- Tkinter (GUI)
- JSON (almacenamiento)
- PyInstaller (ejecutable)

## Instalacion desde codigo
```bash
git clone https://github.com/Lesternx/save-pass.git
cd save-pass
pip install -r requirements.txt
python passwork_generator.py
```
